Digestive tract Adventure

Overview

Digestive tract Adventure is a short platform game of N level.The core game mechanic is the player need to control the ninja to avoid attach and obstacle to reach the destination to escape out from the Digestive organs

It is a level design course project.l finish it with my classmates Sandra.Sandra is the main designer.l focus more on testing.

The game engine we choose is Ned.Ned is built-in level editor of game N.N is an award-winning 2d action platformer game. In N, the player controls a ninja who navigates tile-based levels while simultaneously avoiding hazards, collecting gold, and eventually opening an exit door which completes the level.
