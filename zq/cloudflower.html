Puzzle Game:Cloud and Flower

Overview

Cloud and flower is a 10 level puzzle game.The core game mechanic is the player need to control the cloud to combine the flower or flower blocks in a right way,so the cloud is able to arrive the cloud destination.

It is a game assets course project.l finish it independently,from game design,making game assets,game development,preparing the promotion materials to launch it on the itch.

The game engine l choose to make this puzzle game is PuzzleScript.PuzzleScript is an open-source HTML5 puzzle game engine which can develop the puzzle game very fast.

      🕹 Independent Development
      💻 PuzzleScript

Design Process

    1.write one page game design document
    2.decide the game aesthetics and moodboard
    3.game character design
    4.create game environments
    5.level design
    6.prototype
    7.sound design(use built-in sound of PuzzleScripts)
    8.prepare the promotion materials
    9.publish and promote the game(publish on itch)
