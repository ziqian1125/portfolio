Design Process

Our game design team collabrated with the teaching team,the teaching team will first provide us with the Mathematical topic and knowledge. Based on the knowledge,we will brainstom the game ideas which help kids master those knowledge.After the brainstom,we decide the game Core Mechanics and themes.Each game designer will be responsive for writing the game design documents for the specific knowledge.l also need to draw the low or high-fi prototype for game.

After finishing the game design documents and prototypes ,our team will have a meeting with art,animation and teaching teams to discuss the initial design. And their opinions were recorded in the files,we will iterate the game design documents and prototypes based on their feedback. And after that,the art team will design the initial game arts.Our team needs to review the initial game arts with the teaching team and give our opinions. After the second version of game arts are finished,the animation team will begin to design the animation for game.

If the current game editor doesn't have the game functions we need,we also need to write the development requirements documents and submit the files to development team.

l also need to submit game audio and music needs to the game music design team,and review or edit these sound effects and musics.

If the game arts,animation and game editor functionality are all set,l will begin to configure the game using Cocos Creator.After finishing that,l will upload all the game assets and codes to the company's game platform.

Our team leader and testing team will be responsible for testing the game in the different testing environments.They will give me the feedback,l need to fix these bugs.

Iteration

After the project is shipped,the user research team will test the game with some targeted kids and parent.They will write a testing report based on the observations and interviews. They will share the testing reports with all related teams. The game design team will iterate the game based on the results.

This game aims to teach the kids the concept of periodic law.
