A 2D Game Dev Team-Project:Caged

Overview

Caged is a short 2D Platformer game demo, in which the protagonist, being held captive for entertainment purposes, has to escape on a day when the caretaker forgets to close the cage.

The idea was to raise awareness about animals in captivity by having the player experience their situations.

l am the lead programmer of the game,mainly responsible for writing the balance system and winning or losing condition of the game. l am also responsible for merging and reviewing code for other teammates.l also help teammates to fix the bugs when they meet some problems. l was also the project manager to collabrate with teammates to ship the project on time.

Game Rules

The player needs to control the main character to escape the cage.In this process,he needs to prevent the health,entertainment and starvation value from dropping to 0.If any of these values drop to 0.The game will end.

The entertainment value and starvation will be descreased 1 points each 30 seconds.Player can control the character to entertain the animals to recover the values. The starvation value will be descreased 1 points if the player was hurt by the bricks or spikes.

The main challenges for the player including:Keeping up people’s happiness, managing stamina, completing obstacle course in limited time and escaping zoo.
